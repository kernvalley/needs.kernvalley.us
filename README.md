## Installation
1. Fork on [GitHub](https://github.com/kernvalley/needs.kernvalley.us)
2. `git clone ${fork_url}`
3. `cd` into cloned directory/folder
4. `npm i`

## Requirements
- [NodeJS](https://nodejs.org/) v10.16.3
